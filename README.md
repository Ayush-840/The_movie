# CineVault — Movie Watchlist App

## Overview

CineVault is a movie search and watchlist web application built using JavaScript. Users can search for movies, view details, and save them to a personal watchlist.

---

## API Used

OMDb API
https://www.omdbapi.com/

---

## Features

* Live movie search using API
* Display movie poster, title, year, and rating
* Add and remove movies from watchlist
* Watchlist stored using localStorage
* Empty state messages for no results

---

## Technologies Used

* HTML
* CSS
* JavaScript
* Fetch API
* Local Storage

3. Replace API key in config.js

4. Open index.html in browser
